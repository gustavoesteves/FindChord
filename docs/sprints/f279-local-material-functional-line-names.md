# F279 - Nomes funcionais para linhas sugeridas

## Objetivo

Trocar nomes de linhas sugeridas que dependiam de escola, autor ou estilo por descricoes do gesto musical.

## Alteracoes

- Linhas sugeridas passam a usar nomes como `Contorno maior descendente`, `Ascensão pela #11` e `Dominante com aproximação`.
- A dica dorica foi neutralizada para enfatizar a 6ª maior sem depender de exemplos estilisticos.
- A descricao de clima da escala bebop dominante passa a falar em linearidade cromatica.

## Resultado

As linhas continuam musicalmente reconheciveis, mas a UI fica mais coerente com a proposta de guiar o compositor pela funcao do material.
