# F182 - Linguagem de UI para materiais do acorde

## Objetivo

Reduzir a leitura de “escala compatível” como catálogo abstrato e aproximar a UI do que importa para compositor, arranjador e improvisador: material tocável, célula curta e resolução.

## Mudanças

- A aba do `Escrever` passa de `Escalas Compatíveis` para `Materiais do acorde`.
- O painel do `Escrever` passa a falar em escalas, células e braço para o acorde selecionado.
- O `Harmonizar` passa a destacar o primeiro material melódico diretamente na leitura do acorde, antes do detalhe expandido.
- Células e resoluções deixam de aparecer apenas como texto corrido dentro de `Ver leitura`.

## Critério

A escala continua existindo como fonte, mas a UI deve privilegiar o material musical praticável. Quando houver contexto, a resolução precisa aparecer como parte da ideia, não como dado secundário.
