# F183 - Materiais modais estáveis

## Objetivo

Completar o primeiro ciclo de vocabulário melódico com materiais que não dependem necessariamente de dominante ou resolução cromática.

## Implementação

- `m7` com leitura dórica recebe `m7 dórico / 6`.
- `maj7` com leitura lídia recebe `maj7 lídio / tríade do II`.
- `sus` e `7sus4` com leitura mixolídia recebem `dominante sus / pentatônica`.
- Quando `7sus4` abre para o mesmo dominante, a quarta suspensa explicita a resolução para a terça.

## Exemplos

`Dm7`

```text
D-F-A-C
D-F-A-B
E-F-D
```

`Cmaj7`

```text
C-E-G-B
D-F#-A
F#->G
```

`G7sus4 -> G7`

```text
G-C-D-F
A-C-D-G
C->B
```

## Critério musical

Esses materiais são úteis porque oferecem vocabulário tocável em acordes estáveis, predominantes ou suspensos. A escala continua existindo como mapa, mas a UI passa a privilegiar células com função musical clara.
