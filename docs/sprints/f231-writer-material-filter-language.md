# F231 - Linguagem dos filtros de material

## Objetivo

Deixar a legenda do fretboard de materiais mais leve para leitura musical imediata.

## Alterações

- `Nota Modal` passou a ser exibido como `Cor`.
- `Evitar (Avoid)` passou a ser exibido como `Evitar`.

## Resultado

Os filtros mantem o mesmo significado interno, mas a UI fica menos tecnica e mais escaneavel.
