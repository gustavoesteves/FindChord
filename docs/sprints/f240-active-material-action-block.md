# F240 - Tocar agora como bloco de acao

## Objetivo

Aproximar a tela Materiais do acorde do desenho conceitual de navegacao musical: primeiro o compositor escolhe uma intencao, depois toca uma ideia concreta e so entao aprofunda vocabulario e mapa teorico.

## Alteracoes

- O bloco `Tocar agora` saiu do painel lateral.
- A ideia inicial virou um bloco de acao proprio entre o acorde/rotas e a grade de materiais.
- O painel lateral passa a focar `Vocabulario util` e `Mapa de apoio`.
- O fluxo visual fica mais proximo de: acorde -> rota -> tocar -> explorar -> ver no braco.

## Resultado

A tela deixa de tratar a frase de estudo como detalhe explicativo e passa a apresenta-la como gesto musical inicial. Isso ajuda o modulo Escrever a servir melhor ao compositor/arranjador: menos leitura de catalogo, mais convite imediato para experimentar uma sonoridade.
