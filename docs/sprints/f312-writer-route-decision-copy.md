# F312 - Texto de decisao nas rotas de materiais

## Objetivo

Deixar claro que as rotas de `Materiais do acorde` representam escolhas musicais, nao apenas filtros de lista.

## Alteracoes

- Cada card de rota voltou a exibir sua descricao curta.
- A descricao permanece tambem disponivel no modelo como parte do contrato da rota.

## Decisao

No `Escrever`, o compositor esta diante de um acorde isolado. A UI precisa explicitar a intencao local: repousar, colorir, tensionar ou sair e voltar.

## Validacao

- `npm run build`
