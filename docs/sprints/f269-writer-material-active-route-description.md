# F269 - Descricao da rota ativa

## Objetivo

Usar a descricao musical da rota sem aumentar a carga visual dos botoes.

## Alteracoes

- O navegador de `Materiais do acorde` passa a exibir a descricao apenas da rota ativa.
- Os botoes continuam compactos, mostrando somente `Ficar dentro`, `Colorir` ou `Tensionar`.

## Resultado

O compositor entende melhor a intencao da rota escolhida sem precisar lidar com tres blocos de explicacao concorrendo entre si.
