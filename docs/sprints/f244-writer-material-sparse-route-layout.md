# F244 - Layout para rotas objetivas

## Objetivo

Melhorar a leitura quando uma rota musical tem poucas ideias disponiveis.

## Alteracoes

- A tela detecta rotas com uma ou duas ideias.
- Nessas rotas, os cards ocupam uma coluna ampla em vez de ficarem presos em uma grade pequena.
- O scroll vertical da grade e removido quando nao ha volume suficiente para justificar lista rolavel.
- A coluna indica `Rota objetiva` para deixar claro que a escassez e uma curadoria, nao uma falha visual.

## Resultado

Quando o acorde oferece poucas escolhas em uma rota, a tela passa a tratar isso como foco musical: menos opcoes, mais clareza.
