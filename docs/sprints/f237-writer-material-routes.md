# F237 - Rotas musicais para materiais

## Objetivo

Comecar a migrar `Materiais do acorde` do modelo de catalogo de materiais para o modelo de navegacao musical do acorde.

## Alterações

- Criado `writerMaterialRoutes`.
- A tela passa a agrupar a paleta em tres rotas:
  - `Ficar dentro`;
  - `Colorir`;
  - `Tensionar`.
- A rota ativa filtra os cards e define o foco automatico inicial.

## Resultado

O compositor passa a escolher uma intencao musical antes de escolher uma fonte teorica. Isso aproxima a tela do prototipo conceitual de navegacao do acorde.
