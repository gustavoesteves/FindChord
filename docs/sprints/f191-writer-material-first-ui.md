# F191 — Materiais musicais antes das escalas

## Objetivo

Consolidar a mudança de linguagem do módulo **Escrever**: a funcionalidade deixa de parecer uma lista de escalas compatíveis e passa a apresentar primeiro o material musical acionável.

## Mudança

- O título principal do card passa a ser o material melódico quando houver vocabulário disponível.
- A escala aparece como **fonte** ou **mapa**, não como decisão principal.
- O painel lateral exibe primeiro os materiais para tocar.
- A leitura teórica da escala foi movida para uma camada secundária chamada **Mapa teórico**.
- O texto vazio agora orienta o usuário a selecionar um material, não uma escala.

## Critério de produto

O compositor/arranjador deve bater o olho e encontrar uma resposta prática:

> Que material posso usar sobre este acorde?

Depois disso, se quiser, ele pode consultar a escala que organiza aquele território no braço.

## Próximo passo

Separar materiais por famílias musicais:

- células;
- arpejos;
- pentatônicas;
- simetrias;
- cromatismos;
- escalas/mapas.

Essa etapa pode transformar o painel em uma ferramenta de vocabulário, deixando a antiga leitura de “escala compatível” como apoio visual.
