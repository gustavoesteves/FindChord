# F263 - Linguagem do bloco tocavel

## Objetivo

Remover termos tecnicos do bloco principal de acao em `Materiais do acorde`.

## Alteracoes

- A frase curada passa a usar o rotulo `Tocar agora`.
- O fallback de notas passa a usar `Notas do material`.
- O botao do fallback troca `Ouvir mapa` por `Ouvir notas`.

## Resultado

O bloco principal diferencia uma ideia melodica curada de uma escuta simples das notas sem expor a nocao interna de mapa/escola.
