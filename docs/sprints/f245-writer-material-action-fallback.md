# F245 - Acao tocavel para todo material ativo

## Objetivo

Manter o fluxo principal da tela mesmo quando um material ainda nao possui frase curada cadastrada.

## Alteracoes

- O bloco `Tocar agora` passa a usar a frase de estudo quando ela existe.
- Quando nao ha frase, o bloco mostra uma audicao curta do proprio mapa/material.
- O botao muda de `Ouvir ideia` para `Ouvir mapa` nesse caso.

## Resultado

Materiais como lidio dominante continuam oferecendo uma acao musical imediata, mesmo antes de termos vocabulario detalhado para eles.
