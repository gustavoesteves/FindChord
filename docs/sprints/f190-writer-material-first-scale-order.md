# F190 — Ordem por materiais no Escrever

## Objetivo

Melhorar a leitura de **Materiais do acorde** no módulo **Escrever**, usando a cobertura melódica já criada para priorizar o que ajuda diretamente o compositor/arranjador.

## Mudança

- A lista de leituras compatíveis agora prioriza escalas que possuem materiais melódicos cadastrados.
- A ordenação usa a quantidade de materiais e a confiança contextual como critério local.
- O card continua mostrando a escala, porque ela ainda é o mapa do braço.
- Quando uma leitura possui mais de um material, o card mostra um indicador compacto com a quantidade adicional.

## Critério musical

No acorde isolado, a interface deve começar pelo vocabulário tocável:

- células;
- arpejos;
- notas-guia;
- pentatônicas;
- estruturas simétricas;
- materiais de tensão/resolução.

A escala permanece como organização visual, mas deixa de ser a primeira camada de decisão musical.

## Próximo passo

Avaliar se o painel deve ganhar uma visão ainda mais direta por material, por exemplo:

- **Células**
- **Arpejos**
- **Pentatônicas**
- **Escalas**

Essa separação pode reduzir ainda mais a sensação de que a funcionalidade é apenas uma lista de escalas.
